# Bahamut/pages/theme

## 概述
theme 模組提供應用程式主題管理功能，支援自訂色彩和主題切換。

## 主要元件

### Theme
主題資料模型，定義主題的色彩配置

### ThemeStore
主題儲存管理器，負責主題的儲存和載入

### ThemeManagerPage
主題管理頁面，提供主題選擇和預覽介面

### ThemeFunctions
主題相關的工具函式

## 功能
- 預設主題
- 自訂主題色彩
- 主題切換
- 主題預覽
- 主題匯入/匯出

## 色彩配置
主題包含以下色彩設定：
- 主色調
- 背景色
- 文字色
- 強調色
- 連結色
- 推文色彩
- 狀態列色

## 主題類型
- 淺色主題
- 深色主題
- 高對比主題
- 自訂主題

## 技術特點
- 動態主題切換
- 本地主題儲存
- 即時預覽
- 使用 Kotlin 開發
