# Bahamut/pages/bbsUser

## 概述
bbsUser 模組提供 BBS 使用者相關功能，包含使用者資訊查詢和設定頁面。

## 主要元件

### UserInfoPage
使用者資訊頁面，顯示使用者的基本資料和統計資訊
- ID
- 暱稱
- 上線次數
- 發文數
- 等級
- 上次上線時間等

### UserConfigPage
使用者設定頁面，管理個人設定和偏好

## 功能
- 查詢使用者資訊
- 寄信給使用者
- 加入好友/黑名單
- 修改個人設定

## 資料來源
- 從 BBS 即時查詢
- Telnet 畫面解析

## 技術特點
- 使用者資料解析
- 互動操作處理
- 使用 Kotlin 開發
