# telnet/logic

## 概述
logic 模組提供 Telnet 相關的業務邏輯處理，包含文章處理、看板搜尋、分類模式等功能。

## 主要元件

### ArticleHandler
文章處理器，處理文章資料的解析和轉換邏輯

### SearchBoardHandler
看板搜尋處理器，處理看板搜尋相關邏輯

### 分類模式
- `ClassMode` - 分類模式定義
- `ClassStep` - 分類步驟

### ItemUtils
項目工具函式，提供列表項目處理的通用方法

## 功能
- 文章內容解析
- ANSI 碼轉換為結構化資料
- 看板列表處理
- 文章列表項目識別
- 分類導航邏輯

## 使用場景
- 解析 Telnet 接收的原始資料
- 識別畫面內容類型
- 提取文章資訊
- 處理看板搜尋結果

## 技術特點
- 資料解析和轉換
- 模式識別
- 使用 Kotlin 開發
