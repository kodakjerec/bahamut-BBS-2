# asFramework/utils

## 概述
utils 模組提供工具類別，包含串流讀寫和型別轉換功能。

## 主要元件

### ASStreamReader
串流讀取器，提供從輸入串流讀取資料的工具方法

### ASStreamWriter
串流寫入器，提供向輸出串流寫入資料的工具方法

### ASTypeConvertor
型別轉換器，提供各種資料型別之間的轉換功能

## 功能
- 檔案和網路串流讀寫
- 資料型別轉換
- 位元組陣列處理
- 字串編碼轉換

## 使用場景
- 網路資料讀寫
- 本地檔案操作
- 資料序列化/反序列化
- 型別轉換

## 技術特點
- 統一的 I/O 介面
- 安全的型別轉換
- 使用 Kotlin 開發
