# asFramework/ui

## 概述
ui 模組提供自訂的 UI 元件庫，包含列表視圖、捲動視圖、通知元件等。

## 主要元件

### ASListView
自訂列表視圖，提供高效的列表顯示功能
- `ASListViewItemView` - 列表項目視圖介面
- `ASListViewOverscrollDelegate` - 過度捲動委派
- `ASListViewExtentOptionalDelegate` - 擴充選項委派

### ASScrollView
捲動視圖，提供內容捲動功能

### 通知元件
- `ASToast` - 快顯訊息
- `ASSnackBar` - 底部通知條

## 功能特點
- 高效的列表渲染
- 自訂捲動行為
- 過度捲動效果
- 使用者通知功能

## 使用場景
- 文章列表顯示
- 看板列表
- 長內容捲動
- 使用者訊息提示

## 技術特點
- 自訂視圖實現
- 高效能渲染
- 靈活的委派機制
- 使用 Kotlin 開發
